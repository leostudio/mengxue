# 萌学汉字 - 设计规格文档

## 概述

在现有萌学乐园应用中新增"萌学汉字"模块，与"萌学字母"和"萌学视力表"平行。用户可以学习 300 个最常用汉字，包含读音播放、相关词语（配真实图片）、笔顺动画和描红练习。目标年龄段 3-6 岁。

## 页面流程

```
主页 → 汉字列表页 → 汉字详情页 → 汉字描红页
```

共新增 3 个页面（screen），主页新增一个入口卡片。

---

## 页面设计

### 1. 主页入口

在现有 `home-grid` 中新增第三个卡片：

- 图标：🀄
- 标题：萌学汉字
- 描述：学习常用汉字

### 2. 汉字列表页 (hanzi-grid-screen)

**布局：**
- 顶部：返回按钮 + 标题 "🀄 萌学汉字 🀄"
- 搜索框：支持汉字和拼音搜索
- 难度筛选标签：全部 / ⭐初级 / ⭐⭐中级 / ⭐⭐⭐高级
- 汉字网格：5 列布局，每个卡片显示汉字 + 拼音

**难度分级规则：**
- 初级（⭐）：1-5 画
- 中级（⭐⭐）：6-9 画
- 高级（⭐⭐⭐）：10 画及以上

**搜索逻辑：**
- 输入汉字：精确匹配
- 输入拼音（含不完整）：前缀匹配

### 3. 汉字详情页 (hanzi-detail-screen)

**布局：**
- 顶部：返回按钮
- 大字展示区：拼音 + 大号汉字
- 操作按钮：🔊 听发音 / ✏️ 开始描红
- 相关词语区：横向滑动的词语卡片列表

**行为：**
- 进入页面时自动播放汉字读音（Web Speech API，zh-CN）
- 点击"听发音"可重复播放
- 点击"开始描红"进入描红页

**词语卡片：**
- 每个汉字配 2-3 个常用词语
- 卡片包含：图片区域 + 词语 + 拼音
- 图片来源：Unsplash Source API（`https://source.unsplash.com/200x200/?{query}`）
- 每个词语预设英文搜索关键词（如 "水果" → "fruit,colorful"）
- 图片加载失败时 fallback 到 emoji

### 4. 汉字描红页 (hanzi-trace-screen)

**布局：**
- 顶部：退出按钮 + 当前笔画指示 + 完成按钮
- 中部：Hanzi Writer 渲染区域（田字格背景）
- 底部：🎬 看笔顺 / 🔄 重写 按钮

**行为：**
- 使用 Hanzi Writer 库的 quiz 模式实现描红
- 进入时自动播放笔顺动画一次
- 用户逐笔描红，宽松判定（`leniency: 1.5`）
- 全部笔画完成后：
  - 弹出小红花🌺庆祝动画
  - 语音播报"太棒了！送你一朵小红花！"
  - 2 秒后自动返回详情页

**小红花庆祝动画：**
- 复用现有 celebration 动画框架
- 增加小红花🌺飘落特效
- 显示文字"太棒了！送你一朵小红花！"

## 技术方案

### 外部依赖

- **Hanzi Writer** — 通过 CDN 引入
  - `<script src="https://cdn.jsdelivr.net/npm/hanzi-writer@3/dist/hanzi-writer.min.js"></script>`
  - 笔画数据从 CDN 按需加载
  - 大小：~30KB（库） + 2-5KB/汉字（按需）

### 汉字数据

300 个最常用汉字，按使用频率排序，内置在 `app.js` 中。

数据结构：
```javascript
const HANZI_LIST = [
  {
    char: '一',
    pinyin: 'yī',
    level: 1,  // 1=初级 2=中级 3=高级
    words: [
      { word: '一个', pinyin: 'yī gè', imgQuery: 'one,single', emoji: '1️⃣' },
      { word: '一起', pinyin: 'yī qǐ', imgQuery: 'together,friends', emoji: '🤝' }
    ]
  },
  // ... 共 300 个
];
```

### 图片方案

- **来源：** Unsplash Source API
- **格式：** `https://source.unsplash.com/200x200/?{imgQuery}`
- **无需 API Key**，直接 URL 访问
- **Fallback：** 图片加载失败时显示预设的 emoji
- **缓存：** 浏览器自动缓存

### 语音方案

复用现有 `speakChinese()` 函数（Web Speech API），与萌学字母/视力表一致。

### 导航集成

- `hideAllScreens()` 中新增 3 个 screen 的隐藏
- 主页新增 `btnHanzi` 点击事件
- 所有返回按钮遵循现有模式
